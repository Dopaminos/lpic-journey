1: Добавить в репозитории информацию для работы с пакетом java
```
apt update
```
2: Установить java со всеми зависимостями
```
apt install openjdk-21-jre-headless
```
3: Удалить java
```
apt remove openjdk-21-jre-headless
```
4: Установить webmin
```
curl-им пакет и apt install ./имяпакета.deb
```
5: Обновить все пакеты ПО
```
apt upgrade
```
