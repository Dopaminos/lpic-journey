# Сокеты

Посмотреть **все listening сокеты** системы и определить:

- какой процесс их открыл
- какой интерфейс слушает соединение
- используется ли TCP или UDP

Найти сервисы, которые слушают:

- `127.0.0.1`
- `0.0.0.0`
- `::`

Определить, в чём различия между этими типами привязки.

# Соединения

Установить несколько сетевых соединений (через `ssh`, `curl`, `wget`).

Используя `ss`, определить для каждого соединения:

- локальный IP и порт
- удалённый IP и порт
- состояние TCP соединения

Закрыть соединения и снова посмотреть вывод `ss`.

Определить:

- какое состояние появляется после закрытия соединения
