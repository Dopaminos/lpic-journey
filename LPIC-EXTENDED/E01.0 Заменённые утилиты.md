# netstat → ss

`ss` (socket statistics) - инструмент, позволяющий смотреть:

- активные сетевые соединения
- listening-порты
- tcp/udp сокеты и их статистику

## почему ss, а не netstat

`ss` работает напрямую с ядром через **netlink API**, благодаря чему может отображать больше информации о сокетах и делает это быстрее.

## полезные аргументы

- `-a` - показать всё  
- `-t` - tcp соединения  
- `-tu` - tcp и udp соединения  
- `-l` - только listening сокеты  
- `-n` - numeric вывод (без DNS-резолвинга)  
- `-p` - показать процессы по PID  
- `-e` - extended информация  
- `-i` - дополнительная tcp-информация  
- `-tulpen` - объединение наиболее полезных аргументов  
- `-s` - статистика сокетов  

## фильтрация вывода

Фильтрация может выполняться несколькими способами:

- встроенная фильтрация  
- через `grep`

пример: ss -tulpen | grep :80

## состояния TCP соединений

У TCP соединений существуют различные состояния:

- `LISTEN` - ожидание соединения  
- `ESTABLISHED` - активное соединение  
- `TIME_WAIT` - соединение закрыто, но порт ещё удерживается системой  
- `CLOSE_WAIT` - соединение ожидает закрытия  
- `SYN_SENT` - запрос соединения отправлен  
- `SYN_RECV` - запрос соединения получен

# ifconfig → ip

`ifconfig` (interface config) использовался для:

- просмотра доступных интерфейсов  
- назначения ip-адресов  
- включения и выключения сетевых интерфейсов  

Современная замена - утилита **`ip`** из пакета **iproute2**.

`ip` объединяет в себе функциональность нескольких утилит:

- `ifconfig`
- `route`
- `arp`
- частично `netstat`

## полезные аргументы

- `ip addr` или `ip a` - посмотреть интерфейсы и назначенные IP  
- `ip link` - показать только интерфейсы  
- `ip link set <интерфейс> up` - поднять интерфейс  
- `ip link set <интерфейс> down` - выключить интерфейс  
- `ip route show` - посмотреть таблицу маршрутизации  
- `ip neigh` - показать ARP-таблицу (таблицу соседей)  
- `ip -s link` - статистика сетевого интерфейса  
- `ip addr add <ip>/<mask> dev <interface>` - добавить дополнительный IP-адрес интерфейсу (временно)  
- `ip monitor` - мониторинг сетевых изменений в реальном времени  
- `ip -br a` - краткий вывод интерфейсов
